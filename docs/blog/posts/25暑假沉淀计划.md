# 放弃幻想，准备斗争

## 人啊，先幻灭你自己！

没有那么完美的计划，没有什么竭力的自我

世界他轻而易举将时间粉碎

本就并非能够将自我完整贯彻的主人公，应知渴望完成的事也并非惊天伟业

身为弱者却还在沉沦在完美主义的臆念中苟且度日

阴沟虫豸仍然妄图自己能像虚枉的童话般美好结局

我们身就不完美的此生，唯有苟延残喘逐其一世

将七零八碎的闪光贯彻星空

我们迷途知返恍然大悟，且将目光指往浩瀚星辰

究其一生的星光为此刻闪烁
